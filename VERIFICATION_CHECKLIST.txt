QUICK FIXES (If Issues)
────────────────────────────────────────────────────────────────────

Issue: Build fails
  → Check: npm run build (locally first)
  → Check: TypeScript errors in terminal
  → Fix: Address compilation issues
  → Re-deploy: Try again

Issue: Logo doesn't show in email
  → Check: /public/logo.png exists
  → Check: Email was actually sent
  → Check: Email client (try different client)
  → Check: Spam folder
  → Check: Error logs in console

Issue: Email not sent
  → Check: Email transporter configured
  → Check: EMAIL_USER and EMAIL_PASSWORD set
  → Check: Gmail app password (if using Gmail)
  → Check: SMTP settings (if using custom SMTP)

Issue: Want to disable base64
  → Edit: src/lib/email-logo-utils.ts
  → Change: useBase64: true → useBase64: false
  → Redeploy: npm run build → deploy
